Automated repository backups

Your code, preserved

GitCask creates immutable, verified backups of your GitHub repositories on a schedule you control. Every branch, every tag, every ref — archived and checksummed in Cloudflare R2.

Open source · MIT licensed

How it works

Four steps from repo to archive

01

Register

Add repositories via the API or CLI. Set your preferred backup interval — from every 5 minutes to once a day.

02

Clone

A dedicated container runs git clone --mirror, capturing every branch, tag, and ref in your repository.

03

Verify

The archive is compressed and a SHA-256 checksum is computed to ensure integrity before upload.

04

Store

Backups and metadata land in Cloudflare R2 — immutable, versioned, and available for restore at any time.

Features

Everything you need, nothing you don't

SHA-256 integrity

Every archive is checksummed at creation and verified on upload. Know your backups are exactly what was cloned.

Configurable schedules

Set per-repo intervals from 5 minutes to daily. Trigger manual backups any time with a single API call.

Full metadata capture

Repository description, topics, visibility, language, and branch info are archived alongside the code.

Automatic retries

Failed jobs retry with exponential backoff — up to 4 attempts. Stuck jobs are detected and reported after 15 minutes.

Retention policies

Time-based or count-based cleanup keeps your storage lean. Old artifacts are pruned automatically.

Job visibility

Track every backup through 6 stages — from cloning to metadata upload. Full run history and error details at a glance.

Quick start

Three commands to your first backup

Register a repository, trigger a backup, and check its status. The CLI wraps the full admin API for scripting and automation.

Backups run on schedule from there — or trigger them manually whenever you need a fresh snapshot. 

# Add a repository
gitcask repos add --owner acme --name api

# Trigger an immediate backup
gitcask repos trigger --id repo_01

# Check recent runs
gitcask runs list --repo repo_01

Stop hoping GitHub never goes down

GitCask is open source and self-hosted. Deploy to your own Cloudflare account in minutes.
