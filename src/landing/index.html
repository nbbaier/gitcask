Your repos, always backed up

gitcask automatically backs up your GitHub repositories to Cloudflare R2. Set a schedule, configure retention, and never worry about losing code again.

Start backing up How it works → 
$ gitcask add --repo owner/my-repo --schedule "0 3 * * *"

Core features

Built on Cloudflare, built to last

Cron scheduling
Per-repository cron schedules with automatic execution. Every backup runs on time through Cloudflare Workers.
R2 object storage
Compressed archives stored in Cloudflare R2. Zero egress fees, globally distributed, always available.
Retention policies
Configurable retention per repo. Old backups are automatically pruned to keep storage costs predictable.
Queue-backed retries
Failed jobs route through Cloudflare Queues for automatic retry. No backup is silently lost.
Job tracking
Full audit trail for every backup run — status, duration, size, and errors. Query via API anytime.
REST + CLI
Manage everything through a clean REST API or the gitcask CLI. Automate with your existing tools.
